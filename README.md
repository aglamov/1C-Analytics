# 1C Analytics

Мобильное приложение для iPhone, которое показывает управленческие показатели из 1С в удобном формате: быстрый обзор, детализация по группам и понятная динамика без необходимости открывать десктопные отчеты.

Первый сценарий проекта - дашборд "Образование" по задаче [SED-4884](https://tracker.yandex.ru/SED-4884). Описание подготовлено на основе документа `Показатели_Образование.docx`.

## Цель

Сделать компактный iOS-клиент для просмотра ключевых образовательных показателей из 1С:

- общее количество обучающихся;
- распределение обучающихся по гражданству: РФ и иностранные граждане;
- количество обучающихся по уровням образования;
- количество обучающихся очной формы обучения.

## Источник данных

Базовый источник для первого дашборда: регистр сведений 1С `(вуз). Обучение`, срез последних.

Показатели группируются по уровням образования:

- БАК - бакалавриат;
- СПЕЦ - специалитет;
- МАГ - магистратура;
- АСП - аспирантура;
- ОРД - ординатура;
- Прочие - остальные значения.

## MVP

Первая версия приложения должна закрывать основной мобильный сценарий:

- экран подключения к серверу 1С или аналитическому API;
- список доступных дашбордов;
- дашборд "Образование" с карточками ключевых показателей;
- детализация показателя по группам;
- выбор даты или периода среза;
- обновление данных по запросу пользователя;
- индикация ошибок подключения, пустых данных и устаревшего среза.

## Технический фокус

Проект планируется как нативное iOS-приложение для iPhone. На старте разумная архитектура:

- SwiftUI для интерфейса;
- async/await для сетевых запросов;
- отдельный слой клиента 1С/API;
- модели показателей, независимые от конкретного транспорта;
- мок-данные для разработки UI без доступа к промышленной базе.
